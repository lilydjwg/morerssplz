本站提供各种转 RSS 服务。已经支持的有：

知乎专栏

将知乎专栏首页地址里的用户名写到 https://rss.lilydjwg.me/zhihuzhuanlan/ 之后即可。如专栏首页是 https://zhuanlan.zhihu.com/dingxiangyisheng，则订阅地址为 https://rss.lilydjwg.me/zhihuzhuanlan/dingxiangyisheng。

知乎专栏链接：

这里还有些好用的小书签。添加为书签之后在知乎专栏页面使用即可： 订阅知乎专栏 订阅知乎专栏到 InoReader

如果你发现访问的文章中的图片没有显示，那是因为知乎做了处理。如果你是火狐用户，可以使用如下方案：

  1. 使用 RefControl 扩展来禁止火狐向知乎图片发送 referrer。
  2. 如果你能访问 Google 或者 CloudFlare，可以使用它们提供的图片代理。在 URL 后边加上 ?pic=google 或者 ?pic=cf 就可以了。

想要评论？来我博客。

知乎动态

就是知乎新版个人页面的动态啦。不过只支持回答和文章两种类型。把个人页面链接里那个 ID 写到 https://rss.lilydjwg.me/zhihu/ 后边就可以了（不要带 people/ 或者 org/ 之类的）。比如这个页面 https://www.zhihu.com/org/jian-dan-xin-li，订阅地址就是 https://rss.lilydjwg.me/zhihu/jian-dan-xin-li。

「知乎专栏」支持的参数，这里也支持。比如处理图片显示问题的 pic 参数。

v2ex 评论

将 v2ex 帖子 URL 里的数字写到 https://rss.lilydjwg.me/v2ex/ 之后即可。

小书签： 订阅 v2ex 帖子评论 订阅 v2ex 帖子评论到 InoReader
